Hoisting is a JavaScript mechanism where variables and function declarations are moved to the top of their scope before code execution.
function setEnd(){
 end=10;
}
setEnd();
console.log(end);
Bu 10 çıktısı veriyor.
function setEnd() {
  var end = 10;
}
setEnd();
console.log(end);
Bu ise hata veriyor

Fonksyonun hoisting özeeliği vardır. Fn nin tanımlanmasından önce veya sonra çağırılması fark etmez.
myFunc();
function myFunc() {
  console.log(myNum);
  var myNum = 30;
}
undefined çıktısı verir çünkü "var myNum" myFunc içinde en üst satıra alınsa bile sadece tanımlanması alınır ama 30 değer ataması en üste alınmaz.
Eğer üstteki "setEnd()" örneğindeki gibi var yazmasaydı tanımlanma; myFunc içinde en üst satıra değilde dosyanın en üst satırlarına alınırdı.
