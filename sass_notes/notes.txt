import ederken dosya önüne tri spnuna .scss koyma. 
use "colors" as c

importla eklenirse variable'lar global oluyor. Farklı scss dosyalarında aynı isimde variable varsa import problem olur. Sonra gelen ezer.
Bu nedenle import yerine use kullan.

mixing vs function farkı: function değer return eder. bu nedenle function hesaplama yapıp değer döndürmek için kullanılır. Function karmaşık yapılar için for döngüsü için felan kullanılır.
eğer css stili döndüreceksen function değil, mixing kullan
